# CoolView

## 说明
使用hexo搭建的博客，运用了`hexo-cool-theme`主题。

## Install
1. 安装`exo-cool-theme`主题
git clone https://github.com/cool1024/hexo-cool-theme.git themes/cool1024
2. 安装依赖
npm i

## 相关操作
1. 开启预览服务器
`npm run serve`
2. 添加新文章
`hexo new 'xx标题'`
3. 打包网站
`npm run build`
